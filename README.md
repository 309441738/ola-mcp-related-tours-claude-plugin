# Related Tour Packages · Claude Code 插件

Explore related land tour packages that complement your cruise experience, enriching your travel journey.

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-mcp-related-tours-claude-plugin
/plugin install ola-mcp-related-tours@ola-mcp-related-tours-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Related Tour Packages**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
